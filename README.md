# pretty-landing

A polished demo marketing landing for a **fictional** product ("Aurora"),
built entirely from [machin-web-ui](https://github.com/javimosch/machin-web-ui)
components + gradient utilities, themed violet via one `theme.json`. Proves the
framework can craft attractive landing pages, not just dashboards.

Not a real product — every logo, quote, and stat is invented.

```sh
MACHIN_WEB_UI=/path/to/machin-web-ui ./build.sh   # needs machin + zig
./app                                             # http://localhost:48100
./export.sh                                       # static -> docs/ (GitHub Pages)
```
