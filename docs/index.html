FAQ

Questions, answered.

Do I need to design anything?+

No. Aurora renders every page and widget from your brand theme — one file, applied everywhere. You never touch CSS.

Can I use my own domain?+

Yes, on Team and up. Point a CNAME and your changelog lives at changelog.yourdomain.com.

Is there a free plan?+

Yes — Starter is free forever, and every plan is free for open-source projects.

How do users see updates in-app?+

A two-line snippet renders a 'What's new' widget; read state is tracked per user so nobody sees the same update twice.
